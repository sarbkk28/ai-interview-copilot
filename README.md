# AI Interview Copilot

An AI-powered interview preparation platform that generates personalized interview questions from a user's resume, evaluates spoken answers using LLMs, and provides detailed feedback and performance reports.

## Tech Stack

- Python
- FastAPI
- Streamlit
- Groq Llama
- Faster-Whisper
- SQLite

## Features

- Resume Upload
- AI Question Generation
- Voice Interviews
- AI Evaluation
- Performance Dashboard
